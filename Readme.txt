Дополнено f-mail:

- Использование программы DisplayFusion для переключения конфигурации мониторов.
- Выход из сессии
- и еще ряд разных изменений.

Оригинал:

Программа быстрого переключения пользователей.

Скомпилированный вариант находится в папке Release.

Установка:
- Запустите "Install Service (Run As Admin).bat" для установки и запуска службы быстрого переключения пользователей (sus.exe)
- Поставьте приложение suc.exe в автозагрузку всем пользователям.

Удаление:
- Запустите "Uninstall Service (Run As Admin).bat" и удалите приложение suc.exe из автозагрузки.

Работа:
Просто нажмите Win+A

Программа поставляется как есть, с исходным кодом, без гарантий и ограничений.
Работоспособность проверена в Windows XP/Vista/7/8
